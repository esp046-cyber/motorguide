# motorguide
2026 Motors Guide mobile app with the following features:
Key Features:
5 Main Sections via tabs: Instructions, Repair, Skills, Size Guide, and Motor Types
Compact Design: Small fonts (0.85-1.4em), 1% margins throughout
Motors Theme: Dark blue/red gradient color scheme with professional styling
Mobile-Optimized: Responsive layout that works perfectly on phones
Content Includes:
Instructions Guide - Pre-operation checklists, startup procedures, performance monitoring
Repair & Maintenance - Step-by-step bearing replacement, winding tests, troubleshooting
Skills Guide - Beginner/Intermediate/Advanced level tasks and required tools
Size Guide - NEMA and IEC frame sizes, power calculations, voltage ratings
Type Guide - Different motor types (AC, PMSM, BLDC, Universal) with specs and applications
The app features color-coded skill levels, warning/tip boxes, numbered steps, and specification grids for easy reference. Everything is touch-friendly and scrolls smoothly on mobile devices.
